# Lichtara Core Architecture

## Sintese

Este documento existe para declarar, em uma pagina, qual e o sistema Lichtara.

Ele nao substitui a [matriz-canonica-do-sistema.md](matriz-canonica-do-sistema.md). Ele a resume em forma de arquitetura nuclear.

## Estrutura Fundamental

O sistema se organiza por tres eixos:

`Field`

`Coherence`

`Limit`

Esses tres eixos aparecem no Lichtara como:

| Eixo | Expressao no sistema | Funcao |
| --- | --- | --- |
| Field | Livro Vivo | abrir o campo da travessia |
| Coherence | Sistema | organizar forma, relacao e consistencia |
| Limit | Governanca | definir borda, criterio e responsabilidade |

## Quatro Expressoes do Mesmo Sistema

Os tres eixos nao ficam abstratos. Eles se tornam legiveis em quatro expressoes:

| Expressao | Papel |
| --- | --- |
| Livro Vivo | narrativa |
| Mandala | cartografia |
| Portal | navegacao |
| Paper | formalizacao |

Formula resumida:

`Livro Vivo -> narrativa da travessia`

`Mandala -> cartografia da travessia`

`Portal -> navegacao da travessia`

`Paper -> formalizacao da travessia`

## Nucleo Operativo

No centro funcional do sistema:

| Agente | Funcao | Pergunta |
| --- | --- | --- |
| NAVROS | orientacao | onde estou |
| SYNTARIS | alinhamento | o que precisa se alinhar |
| FLUX | movimento | qual e o proximo movimento |

Regra de arquitetura:

- NAVROS le a posicao
- SYNTARIS organiza coerencia
- FLUX traduz coerencia em gesto

## Camadas de Desdobramento

Quando o sistema sai do nucleo e se torna experiencia, ele se organiza assim:

| Camada | Estrutura |
| --- | --- |
| Mandala | 16 agentes |
| Ciclos | 4 ciclos |
| Portal | 3 rotas |
| Jornada | 7 etapas |

Isso significa que o portal nao inventa uma jornada arbitraria. Ele traduz em interface uma arquitetura ja presente no sistema.

## Escalas de Organizacao

Uma forma simples de ler a arquitetura inteira e:

| Nivel | Numero | Funcao |
| --- | --- | --- |
| estrutura | 3 | `Field - Coherence - Limit` |
| processo | 4 | percepcao, estrutura, acao e expansao |
| navegacao | 7 | jornada guiada legivel |
| cartografia | 16 | mapa expandido da mandala |

Formula resumida:

`3 principios -> 4 estados -> 7 etapas -> 16 posicoes`
